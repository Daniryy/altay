САЙТ САЛОНА «АЛТАЙ» — как пользоваться
========================================

СОСТАВ ПРОЕКТА
--------------
index.html       — сам сайт (открой в браузере, чтобы посмотреть)
hero.webp        — главная картинка на первом экране (обновлена: цвет-грейд в тон бренду)
og.png           — превью для WhatsApp/Telegram/VK и соцсетей
quote.webp       — фон блока с цитатой
exterior.webp    — фото входа/фасада
g1…g9.webp       — фотографии для галереи

Все файлы должны лежать В ОДНОЙ ПАПКЕ. Ничего переименовывать не нужно.

КАК ОПУБЛИКОВАТЬ
----------------
Загрузи все файлы из этой папки на свой хостинг (или в репозиторий GitHub Pages) —
index.html должен лежать в корне рядом с картинками.

ЧТО ПОМЕНЯТЬ ПОД СЕБЯ (по желанию)
---------------------------------
• Домен. Если адрес сайта будет не daniryy.github.io/altay-massage/, замени этот
  адрес в 6 местах в начале index.html: canonical, og:url, og:image, twitter:image,
  и поля url/image в блоке разметки JSON-LD. Рядом в коде есть комментарий-напоминание.
• Телефон и соцсети. В разделе «Контакты» проверь номер телефона и ссылки
  WhatsApp / Telegram.
• Цены и услуги — при необходимости поправь в разделе «Услуги и цены».

ЧТО ИЗМЕНЕНО В ЭТОЙ ВЕРСИИ
--------------------------
• Новая главная картинка со спокойным кинематографичным цвет-грейдом,
  который сочетается с зелёно-золотой палитрой салона.
• Галерея: при наведении фото мягко приближается, появляется золотая рамка
  и деликатное затемнение под подписью — аккуратнее и «дороже».

Если после замены браузер показывает старое — обнови страницу с очисткой кэша (Ctrl+F5).
